                                                                          In Scala, as with other statically typed functional languages, functions have types. Both f and g must be functions of the right type for flatMap. If you have an instance of a monad with type M[A] then g must be of type A -> M[B]. In other words, it must take a value of type A as an argument and return a value of type M[B] as a return. Similarly, f must be of type B -> M[C].
